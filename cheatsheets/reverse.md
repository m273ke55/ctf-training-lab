# Reverse

file, strings, grep, ltrace/strace, objdump, Ghidra и Python bytecode помогают понять тип файла, строки, системные вызовы, дизассемблирование и байткод. Типичный порядок: `file` → `strings | grep edu_ctf` → запуск в песочнице при необходимости → `objdump`/Ghidra → восстановление проверки.
