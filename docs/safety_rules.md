# Правила безопасности

- Все задачи локальные.
- Никакого вредоносного поведения и скрытого вредоносного кода.
- Никакой персистентности.
- Никаких атак на реальные сервисы и сетевые цели.
- Никаких реальных персональных данных.
- Никаких опасных payload.
- Бинарники должны быть собраны из исходников в `src/`.
- `solve.md` и `flag.txt` не выдаются студентам.
- `dist/` содержит только безопасные публичные артефакты.
- Web-задачи запускаются в изолированных контейнерах, публикуются только на учебных адресах и не содержат реальных данных.
- RCE, XXE, file inclusion и command injection разрешено проверять только внутри предоставленных challenge-сервисов.
