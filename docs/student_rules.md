# Правила для студентов

- Флаг сдаётся в формате `edu_ctf{...}`.
- Использование интернета: настраиваемое решение преподавателя перед стартом.
- Индивидуальная или командная работа: настраиваемое решение преподавателя.
- Запрещены атаки на CTFd, реальные сервисы, чужие компьютеры и попытки получить `flag.txt` неигровым способом.
- Подсказки запрашиваются через преподавателя или CTFd.
- Флаги сдаются только через CTFd-борду.
