# Модель баллов

| Сложность | Баллы |
|---|---:|
| easy | 100 |
| medium | 200 |
| hard-but-fair | 300 |

Подсказки могут стоить баллы: мягкая дешевле, почти раскрывающая дороже. CTFd позволяет менять баллы и стоимость подсказок вручную, поэтому модель является стартовой. Для смешанной аудитории держите несколько easy-задач доступными с начала, а hard-but-fair используйте как вызов для сильных студентов.
