# Решение

## Идея

В большом количестве обычных log-строк спрятано одно debug-событие. Его `payload` закодирован в base64 и содержит флаг.

## Шаги

1. Найти подозрочную строку:

   ```bash
   grep 'debug_note' dist/server.log
   ```

2. Выделить значение после `payload=`.
3. Декодировать его как base64:

   ```bash
   grep 'debug_note' dist/server.log | awk -F'payload=' '{print $2}' | base64 -d
   ```

4. Проверить, что результат имеет формат `edu_ctf{...}`.
5. Альтернативно запустить `src/solve.py`: он ищет base64-кандидаты и печатает найденный флаг.

Финальный флаг: `edu_ctf{logs_tell_story}`.

## Чему учит задача

Задача тренирует анализ логов, поиск аномальных событий и аккуратное декодирование embedded payload.
