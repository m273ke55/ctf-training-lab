# Решение

Флаг спрятан в младших битах пиксельных байтов grayscale PNG.

## Путь через zsteg

Если доступен `zsteg`, можно проверить LSB-каналы:

```bash
zsteg dist/pixels.png
```

Нужная строка имеет формат `edu_ctf{...}`.

## Путь через Python без внешних зависимостей

1. Распарсить PNG chunks и объединить данные `IDAT`.
2. Распаковать `IDAT` через `zlib`.
3. Для каждой строки пропустить первый байт PNG filter.
4. Взять младший бит каждого пиксельного байта.
5. Собрать биты в байты до нулевого байта или закрывающей `}`.

Готовый solver выполняет эти шаги:

```bash
python3 src/solve.py
```

Флаг: `edu_ctf{lsb_pixels_talk}`.
