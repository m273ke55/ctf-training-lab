# Решение

Файл `signal.wav` содержит последовательность устойчивых тонов. Каждый тон соответствует одному символу флага.

## Ручной путь через spectrogram

1. Откройте `dist/signal.wav` в Audacity или Sonic Visualiser.
2. Переключите дорожку в режим spectrogram.
3. Видно, что сигнал состоит из коротких горизонтальных линий на разных частотах.
4. Частоты идут последовательно и кодируют символы из алфавита `abcdefghijklmnopqrstuvwxyz_{}`.
5. Частота символа вычисляется как `800 + index * 40` Гц, где `index` — позиция символа в алфавите.

После восстановления последовательности получается флаг `edu_ctf{spectrogram_says_hi}`.

## Автоматический solver

Solver читает WAV стандартным модулем `wave`, делит сигнал на сегменты, для каждой допустимой частоты считает энергию методом Goertzel и выбирает самый сильный тон.

```bash
python3 src/solve.py
```
