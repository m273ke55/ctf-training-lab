# Решение

## Идея

Vigenere использует повторяющийся ключ: каждая буква ключа задаёт сдвиг для очередной буквы текста. Заметка намекает на английское слово для университетской территории — `campus`.

## Шаги

1. Прочитать `dist/note.txt` и определить ключ `campus`.
2. Прочитать `dist/ciphertext.txt`.
3. Для каждой ASCII-буквы ciphertext взять очередную букву ключа.
4. Выполнить обратный сдвиг:

   ```text
   plain = (cipher - key_shift) mod 26
   ```

5. Символы `_`, `{` и `}` оставить без изменений.
6. Получить строку формата `edu_ctf{...}`.

Готовое Python-решение находится в `src/solve.py`.

Финальный флаг: `edu_ctf{vigenere_bonus}`.

## Чему учит задача

Задача закрепляет Vigenere, циклическое применение ключа и работу с подсказкой, которая не записывает ключ явно.
