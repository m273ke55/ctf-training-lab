# Решение

## Идея

Маршрут `/admin` возвращает `403` для обычных запросов, но отдельно обрабатывает HTTP-метод `HEAD` и помещает флаг в заголовок `X-Flag`.

## Шаги

1. Проверить обычный запрос к `/admin`.
2. Отправить HEAD-запрос и вывести заголовки:

   ```bash
   curl -I http://127.0.0.1:5104/admin
   ```

3. Прочитать значение заголовка `X-Flag`.

## Флаг

`edu_ctf{head_method_reveals_flag}`

## Чему учит задача

При анализе Web-приложения важны HTTP-методы, коды состояния и заголовки, а не только HTML-тело ответа.
