# Бонусная задача Web: Заявление в МФЦ

- **Категория:** Web
- **Сложность:** medium

## Описание

Онлайн-МФЦ принимает только DOCX-заявления и обрабатывает их XML-содержимое. Подготовьте документ, который раскроет локальный флаг.

## Формат флага

`edu_ctf{...}`

## Важно

Работайте только с выданным адресом учебного сервиса. Не загружайте реальные документы или персональные данные.
