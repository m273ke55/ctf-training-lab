# Загрузка с продолжением

- **Категория:** Web
- **Сложность:** hard-but-fair

## Описание

Сайт разрешает загружать изображения и выбирать внутренние страницы. Объедините две особенности приложения и получите флаг.

## Формат флага

`edu_ctf{...}`

## Важно

Работайте только с выданным адресом учебного сервиса. Не загружайте реальные или чувствительные файлы.
