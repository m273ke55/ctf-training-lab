# Решение

## Идея

Checker сравнивает не сам флаг, а результат XOR-преобразования каждого символа с массивом чисел. Чтобы получить флаг, нужно применить XOR ещё раз тем же ключом.

## Шаги

1. Открыть `dist/checker.js`.
2. Найти строку с `const KEY = ...`.
3. Найти массив `const DATA = [...]`.
4. Для каждого числа выполнить `value ^ KEY`.
5. Преобразовать числа обратно в символы и соединить строку.

Пример фрагмента:

```python
flag = "".join(chr(value ^ KEY) for value in DATA)
print(flag)
```

Готовый solver находится в `src/solve.py` и извлекает константы из JavaScript-файла без Node.js.

Финальный флаг: `edu_ctf{js_reverse_win}`.

## Чему учит задача

Задача тренирует чтение checker-кода и восстановление проверяемой строки по обратимой операции XOR.
