# Бонусная задача Reverse: JavaScript checker

Вам выдан `checker.js` — JavaScript checker для проверки флага. Правильный флаг не хранится в файле открытым текстом.

Прочитайте код, найдите `KEY` и массив чисел `DATA`, затем восстановите строку, которую checker ожидает на входе. Node.js для решения не нужен: достаточно понять обратную операцию.
