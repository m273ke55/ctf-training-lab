# Подсказки

1. **Подсказка 1 — мягкая.** `.pyc` можно анализировать без запуска: это скомпилированный Python bytecode с code object внутри.
2. **Подсказка 2 — инструмент или подход.** Используйте `marshal` для чтения code object и `dis` для просмотра байткода.
3. **Подсказка 3 — почти раскрывает метод.** Найдите массив чисел, `XOR_KEY` и `SHIFT`, отмените сдвиг и XOR, затем разверните строку обратно.
