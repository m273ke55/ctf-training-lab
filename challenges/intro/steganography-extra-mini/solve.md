# Решение

## Идея

Флаг не спрятан в звуке. Он находится в текстовой metadata/comment области WAV/RIFF-контейнера.

## Шаги

1. Проверить файл базовыми инструментами:

   ```bash
   file dist/comment.wav
   strings dist/comment.wav
   ```

2. В выводе `strings` найти текстовый комментарий.
3. Найти в комментарии строку формата `edu_ctf{...}`.
4. При автоматизации можно читать файл как bytes и искать флаг регулярным выражением.

Пример:

```python
import re
data = open("dist/comment.wav", "rb").read()
print(re.search(rb"edu_ctf\{[^}]+\}", data).group(0).decode())
```

Финальный флаг: `edu_ctf{intro_wav_comment}`.

## Чему учит задача

Задача показывает, что Steganography иногда начинается с простого осмотра контейнера и поиска служебных текстовых областей, а не со сложного анализа сигнала.
