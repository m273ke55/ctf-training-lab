# Демонстрация Web

Демонстрационная задача первой пары показывает главный риск выполнения пользовательского кода на сервере. Преподаватель запускает сервис через `docker-compose.web.yml`, отправляет безопасный Python-фрагмент и показывает, почему даже простой доступ к файловой системе раскрывает `/flag.txt`.

Сервис является намеренно уязвимым и доступен только в изолированном учебном контейнере.
