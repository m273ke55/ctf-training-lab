# Подсказки

1. **Подсказка 1 — мягкая.** Посмотрите, что это за файл и есть ли внутри читаемые фрагменты текста.
2. **Подсказка 2 — инструмент или подход.** Используйте `file`, затем `strings`; для фильтрации подойдёт `grep`.
3. **Подсказка 3 — почти раскрывает метод.** Выполните `strings` для файла из `dist/` и найдите строку, которая соответствует формату `edu_ctf{...}`.
