# Демонстрация Reverse

Демонстрационная задача первой пары показывает безопасный старт Reverse-анализа: определить тип файла командой `file`, извлечь читаемые строки командой `strings` и не запускать неизвестный артефакт без необходимости.

Флаг демонстрации: `edu_ctf{reverse_demo_strings}`. Публичный файл `dist/demo_blob.bin` создаётся через `src/generate.py` и не хранится в git.
