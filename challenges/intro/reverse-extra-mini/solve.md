# Решение

## Идея

Checker не нужно воспринимать как неизвестный бинарник. Это обычный Python-код, где проверка флага описана явно.

## Шаги

1. Открыть `dist/tiny_checker.py` и найти `SHIFT`.
2. Найти массив `DATA`.
3. Понять, что каждый элемент массива получен как `ord(ch) + SHIFT`.
4. Для каждого числа выполнить обратную операцию: `chr(value - SHIFT)`.
5. Соединить символы в строку.

Пример фрагмента:

```python
flag = "".join(chr(value - SHIFT) for value in DATA)
print(flag)
```

Готовая автоматизация есть в `src/solve.py`.

Финальный флаг: `edu_ctf{intro_reverse_logic}`.

## Чему учит задача

Задача показывает базовый Reverse-подход: читать логику проверки и восстанавливать данные обратной операцией, не запуская неизвестный код без необходимости.
