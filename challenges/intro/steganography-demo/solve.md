# Решение

Флаг находится не в пикселях изображения, а в служебной текстовой части PNG-файла.

## Шаги

1. Выведите читаемые строки:

```bash
strings dist/picture.png
```

2. Найдите строку формата `edu_ctf{...}`.

3. Полученный флаг: `edu_ctf{stego_demo_comment}`.

## Проверка solver-скриптом

```bash
python3 src/solve.py
```
