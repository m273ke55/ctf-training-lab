# Демонстрация Steganography

Демонстрационная задача показывает, что PNG-файл может хранить не только пиксели, но и служебные текстовые chunks. Преподаватель может использовать её, чтобы показать разницу между обычным просмотром картинки и анализом контейнера через `strings` или просмотр PNG chunks.

Публичный файл `dist/picture.png` создаётся скриптом `src/generate.py` и не хранится в git.
