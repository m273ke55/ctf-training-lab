# Решение

## Идея

Файл назван `unknown.dat`, но настоящий формат определяется не расширением, а содержимым и magic bytes.

## Шаги

1. Проверить тип файла:

   ```bash
   file dist/unknown.dat
   ```

2. Убедиться, что это ZIP-архив, несмотря на расширение `.dat`.
3. Посмотреть список файлов внутри архива:

   ```bash
   unzip -l dist/unknown.dat
   ```

4. Извлечь или вывести `note.txt`.
5. Найти строку формата `edu_ctf{...}`.

То же можно сделать Python-скриптом через стандартный модуль `zipfile`; пример находится в `src/solve.py`.

Финальный флаг: `edu_ctf{intro_zip_magic}`.

## Чему учит задача

Задача тренирует проверку настоящего типа файла по содержимому и аккуратную работу с контейнерами, даже если расширение вводит в заблуждение.
