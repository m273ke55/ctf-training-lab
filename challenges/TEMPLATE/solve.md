# Решение для преподавателя

## Идея

Кратко опишите скрытую технику.

## Шаги решения

1. Первый наблюдаемый признак.
2. Команда или инструмент.
3. Извлечение флага.

## Команды

```bash
file dist/example.txt
strings dist/example.txt
```

## Объяснение

Почему эти шаги приводят к флагу.

## Флаг

`edu_ctf{example_flag}`

## Типичные ошибки студентов

- Сразу использовать сложные инструменты.
- Игнорировать условие.

## Как давать подсказки

Начинайте с направления, затем называйте инструмент, затем метод.
