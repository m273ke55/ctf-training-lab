# Подготовка окружения студента

## Минимальный набор

| Инструмент | Зачем нужен |
|---|---|
| Python 3 | Писать простые декодеры, solver-скрипты и проверять байты. |
| Linux shell, WSL или macOS Terminal | Запускать команды и работать с файлами. |
| `file` | Определять тип файла по сигнатурам. |
| `strings` | Искать читаемые строки в бинарниках и артефактах. |
| `xxd` | Смотреть hex-представление файла. |
| `unzip` | Распаковывать zip-архивы. |
| `zip` | Создавать архивы на workshop. |
| `grep` | Фильтровать текстовый вывод. |
| `sed` | Быстро заменять и извлекать текст. |
| `awk` | Обрабатывать колонки и строки. |

## Рекомендуемый набор

| Инструмент | Зачем нужен |
|---|---|
| `binwalk` | Искать вложенные данные и сигнатуры в файлах. |
| `exiftool` | Читать и изменять метаданные изображений. |
| Wireshark | Графический анализ PCAP. |
| `tshark` | Консольный анализ PCAP. |
| Ghidra | Декомпиляция и анализ бинарников. |
| CyberChef | Быстрые преобразования encoding/crypto. |
| `zsteg` | Проверка LSB-данных в PNG/BMP. |
| `steghide` | Работа с классическими stego-контейнерами. |
| Audacity или Sonic Visualiser | Просмотр аудио и спектрограмм. |

## Общие рекомендации

Используйте пакетный менеджер вашей ОС. На Windows удобнее WSL. Не запускайте неизвестные исполняемые файлы вне учебной среды и не используйте реальные сервисы как цели.
