# Заметки по импорту

`challenge.yml` — удобный внутренний формат репозитория, а не гарантия автоматического импорта. Конкретный CTFd может требовать ручной настройки полей, подсказок, файлов, тегов и URL сервисов. Поле `service` описывает тип и внутренний порт Web-задачи; внешний адрес задаёт инфраструктура после развертывания контейнера.
