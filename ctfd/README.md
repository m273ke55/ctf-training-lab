# Перенос задач в CTFd

Для каждой задачи перенесите название, категорию, описание, баллы, флаг, подсказки и публичные файлы из `dist/`. Файлы `solve.md` и `flag.txt` нельзя загружать как публичные: они раскрывают решение и флаг. `challenge.yml` можно использовать как внутреннюю карточку при ручном создании задачи.
